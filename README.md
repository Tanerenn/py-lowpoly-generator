This project converts standard images into geometric low-poly vector art (SVG) using Python.

## Features
- **Edge Detection:** Uses Harris Corner Detection to find key features in the image.
- **Delaunay Triangulation:** Connects points to create a geometric mesh.
- **Vector Output:** Generates high-quality SVG files suitable for plotting or graphic design.
